<objective>
Create the ThemeColors type system and refactor all canvas rendering functions to accept theme colors as parameters, removing every hardcoded color constant from the render pipeline.

Purpose: This is the foundation for light/dark theme support. Canvas cannot read CSS custom properties, so renderers must receive a ThemeColors object. Every downstream plan depends on this color parameterization.
Output: ThemeColors interface with DARK_THEME and LIGHT_THEME objects, and all render functions parameterized by theme colors.
</objective>
