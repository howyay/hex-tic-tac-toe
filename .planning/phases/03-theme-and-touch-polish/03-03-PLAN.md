Add touch state variables to the `<script>` block:
```typescript
let touchStartTime = 0;
let touchStartX = 0;
let touchStartY = 0;
let touchMoved = 0;
let activeTouchId: number | null = null;
let initialPinchDist = 0;
let lastPinchMidX = 0;
let lastPinchMidY = 0;

const TOUCH_TAP_TIME = 200;   // ms (D-09: <200ms)
const TOUCH_TAP_DIST = 10;    // px cumulative (D-09: minimal movement)
```

**handleTouchStart(e: TouchEvent)** (per D-09, D-10, D-11, Pitfall 1):
- Call `e.preventDefault()` to suppress synthesized mouse events and 300ms click delay
- If `e.touches.length === 1`: record `activeTouchId = t.identifier`, `touchStartTime = performance.now()`, `touchStartX = t.clientX`, `touchStartY = t.clientY`, `touchMoved = 0`
- If `e.touches.length === 2`: calculate `initialPinchDist = Math.hypot(t2.clientX - t1.clientX, t2.clientY - t1.clientY)`, record `lastPinchMidX/Y`, set `activeTouchId = null` (cancel tap/drag)

**handleTouchMove(e: TouchEvent)** (per D-10, D-11, Pitfall 2, Pitfall 5):
- If `e.touches.length === 2`: call `e.preventDefault()` to prevent browser pinch zoom. Calculate new distance and midpoint. Compute scale ratio `dist / initialPinchDist`. Get canvas-relative midpoint via `canvas.getBoundingClientRect()`. Call `zoomAtPoint(state.camera, canvasPoint, 1 - scale)` and update `initialPinchDist = dist`. Pan by midpoint delta: `panCamera(state.camera, panDx, panDy)`, update `lastPinchMidX/Y`.
- If `e.touches.length === 1 && activeTouchId !== null`: find touch by `identifier` (Pitfall 5 -- don't use array index). Accumulate `touchMoved += Math.abs(dx) + Math.abs(dy)`. If `touchMoved > TOUCH_TAP_DIST`, pan: `panCamera(state.camera, dx, dy)`, update `touchStartX/Y`.
- Check `gameState.status !== 'won'` before applying pan/zoom (board freeze on win).

**handleTouchEnd(e: TouchEvent)** (per D-09, Pitfall 3):
- If `activeTouchId === null`: return (was a pinch, not a tap)
- Compute `elapsed = performance.now() - touchStartTime`
- If `elapsed < TOUCH_TAP_TIME && touchMoved < TOUCH_TAP_DIST && gameState.status === 'playing'`:
  - Get canvas rect: `const rect = canvas.getBoundingClientRect()`
  - Compute canvas offset: `offsetX = touchStartX - rect.left`, `offsetY = touchStartY - rect.top` (Pitfall 3: must subtract rect)
  - Convert to world: `screenToWorld({ x: offsetX, y: offsetY }, state.camera)`
  - Convert to hex: `pixelToHex(worldPoint, HEX_SIZE)` then `hexRound()`
  - Place stone: `gameState.placeStone(hex)`
- Reset `activeTouchId = null`

**Canvas element** -- add touch event handlers alongside existing mouse handlers:
```svelte
<canvas
  bind:this={canvas}
  onmousedown={handleMouseDown}
  onmousemove={handleMouseMove}
  onmouseup={handleMouseUp}
  onmouseleave={handleMouseLeave}
  onwheel={handleWheel}

></canvas>
```

**Important considerations:**
- Do NOT clear `state.hoveredHex` on touch interactions (it's irrelevant -- touch has no hover state)
- Pan during pinch should also be gated on `gameState.status !== 'won'`
