<task type="checkpoint:human-verify" gate="blocking">
  <name>Task 2: Visual verification of complete hex grid engine</name>
  <what-built>
    Complete Phase 1 hex grid engine with all GRID requirements:
    - GRID-01: Hex grid rendered on canvas with pointy-top orientation and axial coordinates
    - GRID-02: Unbounded grid -- can pan infinitely in any direction
    - GRID-03: Viewport culling -- only visible hexes rendered
    - GRID-04: Click-drag panning with grab/grabbing cursor
    - GRID-05: Scroll-wheel zoom anchored to cursor position
    - GRID-06: Hover preview showing semi-transparent fill on hex under cursor
    - D-02: Subtle grid outlines
    - D-04: Edge fade gradient at viewport borders
    - D-07: LOD dot rendering at far zoom
    - D-11: Debug coordinate overlay via Ctrl+D
  </what-built>
  <how-to-verify>
    Start dev server: `npm run dev` (opens http://localhost:5173)

    1. **Grid renders (GRID-01):** Verify hex grid visible with pointy-top hexes (vertices pointing up, not edges). Outlines should be subtle, semi-transparent white on dark background.

    2. **Pan (GRID-04):** Click and drag -- grid should pan smoothly. Cursor changes from "grab" to "grabbing" during drag. Pan in all directions -- no boundary, no snapping.

    3. **Zoom (GRID-05):** Scroll wheel up = zoom in, scroll down = zoom out. Place cursor over a specific hex, zoom in -- that hex should stay under cursor (Google Maps style). Verify zoom range: can't zoom infinitely in either direction.

    4. **Viewport culling (GRID-03):** Zoom out as far as possible. Grid should still render smoothly (60fps, no jank). At max zoom-out, hexes switch to small dots (LOD).

    5. **Hover preview (GRID-06):** Move cursor over hexes (without clicking). The hex under cursor should fill with semi-transparent blue. Moving to a different hex should instantly update. Moving cursor off canvas should clear the highlight.

    6. **Edge fade (D-04):** Look at viewport edges -- grid outlines should fade to background color near edges, creating an "infinite" feeling.

    7. **Debug mode (D-11):** Press Ctrl+D (or Cmd+D on Mac). Small coordinate labels "(q,r)" should appear at each hex center. A hint "Ctrl+D: coordinates" should appear bottom-left. Press again to toggle off. Verify (0,0) is near center of initial view (D-08).

    8. **Overall feel:** Does the grid feel like "navigating a map" (user's words)? Smooth pan, smooth zoom, responsive hover.
  </how-to-verify>
